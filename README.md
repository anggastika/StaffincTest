### Getting Started

```

Install the dependencies:
```bash
npm install (it will create node_modules folder in root project and download all required dependencies)
```

Run tests:
```bash
npm test (all test)
npm test -- --spec ./features/<featurefilename>.feature   (run specific test)
```
